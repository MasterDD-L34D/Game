# EvoTactics Developer Kit (PTPF-Compatible)

![Status](https://img.shields.io/badge/status-active-green)
![License](https://img.shields.io/badge/license-MIT-blue)

📦 Questo repository integra strumenti evoluti per:
- Validazione YAML e JSON (telemetria & biomi)
- Rilevamento drift semantico (hook pre-commit)
- Valutazioni moduli con ASCII Matrix
- Auto-scoring via AI logic (auto_eval)

🧠 Basato su PrimeTalk PTPF e EchoWake 2.0

## 📁 Struttura Principale
```
/scripts/            → Tool Python e hook JS
/tools/devkit/       → Valutazioni e metodi ausiliari
/telemetry/          → Dataset YAML evolutivo
/schemas/            → JSON schema per validazioni
/docs/               → Note, template, design system
```

## 🚀 Setup Rapido
```bash
git clone <repo>
cd repo
bash installer.sh
```
