Integrazione con il repository

Gli strumenti riutilizzano gli stessi dataset versionati in packs/evo_tactics_pack/ per evitare duplicazioni. Aggiornare il pack garantisce automaticamente contenuti aggiornati anche per il sito.

Pipeline di validazione

I workflow CI e la CLI condivisa eseguono i validatori del pack: eventuali errori fermano la pubblicazione e mantengono sincronizzati dataset e report.

Dataset condivisi

Le pagine qui pubblicate puntano direttamente ai file YAML nel repository. Per distribuzioni esterne è sufficiente servire le stesse directory del pack o sostituire gli URL con un endpoint CDN.
