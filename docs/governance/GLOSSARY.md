# Glossary IT+EN

Termini canonici da usare nei documenti attivi.

| Term              | Definizione breve                                                      |
| ----------------- | ---------------------------------------------------------------------- |
| Workstream        | Flusso operativo con owner, dipendenze e deliverable misurabili.       |
| Source of truth   | Documento primario da cui derivano decisioni operative.                |
| Dual-track        | Modello con canale canonico e canale storico in parallelo.             |
| Governance check  | Verifica automatica metadata/registry/coerenza docs.                   |
| Stale document    | Documento oltre il ciclo di revisione dichiarato.                      |
| Warning-only gate | Controllo CI non bloccante usato nella fase di transizione.            |
| Required gate     | Controllo CI bloccante richiesto per merge in PR.                      |
| Incomplete module | Modulo senza baseline minima (README, ownership, test/smoke, runbook). |
