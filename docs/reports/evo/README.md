# Report integrazione Evo-Tactics

I report generati durante i batch `species_ecotypes` e `traits` verranno
conservati qui. Utilizzare questa cartella per:

- `species_summary.md` e log di validazione (`reports/incoming/` continuerà a
  ospitare i log grezzi).
- `traits_anomalies.csv` e analisi complementari.
- Eventuali changelog riepilogativi per la governance del pacchetto.

Aggiornare l'indice dei report in `reports/data_inventory.md` quando nuovi file
vengono aggiunti.
